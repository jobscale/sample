よくあるご質問

過去に国民年金保険料の未納があっても国民年金基金に加入できますか？

過去に国民年金保険料の未納（一部を含む免除・学生納付特例・納付猶予）があっても、基本的に現在国民年金保険料を満額納付していれば加入できます。

全文を読む
会社を辞めて自営業として独立した場合、いつから加入できますか？

自営業やフリーランスになってから最初に国民年金保険料を納めた月から国民年金基金に加入ができます。国民年金保険料は金融機関・郵便局・コンビニエンスストア等で納付します。

全文を読む
パートタイマーや主婦でも加入できますか？

60歳未満で国民年金の第一号被保険者の方、60歳以上65歳未満で国民年金の任意加入者の方、この要件を満たし国民年金保険料を満額納付されていればご加入いただけます。

全文を読む
加入した後に会社を立ち上げた場合や、サラリーマンに転職等した場合は、それまで預けたお金はどうなりますか？

加入後に法人成りや会社員となって厚生年金保険料を支払うことになった場合には、国民年金基金の加入資格を喪失することになります。ただし、今までの掛金は将来の年金額に反映されます。

全文を読む
全国国民年金基金と職能型基金（歯科医師・司法書士・弁護士）には両方加入できますか？

いずれか一方の基金にしか加入できません。制度上に違いはなく、任意で全国基金から職能型基金、職能型基金から全国基金に移ることもできません。なお、ほけんの窓口では全国基金のお取扱いのみとなります。

全文を読む

国民年金付加保険料を納めています。加入できますか？

付加年金にご加入されている方でも、国民年金基金にご加入いただけます。国民年金基金に加入する場合は付加年金を納める必要がなくなります。

全文を読む
国民年金基金って聞いたことはあるけど、内容がわからないのですが・・・？

全国のほけんの窓口にて、国民年金基金について各種保険と比較しながらご案内ができます。まずは、お気軽にお問い合わせください。

全文を読む
ほけんの窓口で国民年金基金のお申込み手続きはできますか？

直営店は三井住友信託銀行の併営業務代理店として、国民年金基金のお申込みが可能です。パートナー(フランチャイズ)店は全国国民年金基金へのご紹介取次となります。

商売は順調だけど、税金を抑えるには？

「確定申告時の税金をなんとかしたい」という悩みを聞いてくれて助かりました

独立して３年経ち、最近やっと店舗の業績も上向いてきました。
ただ昨年の確定申告時の税金の多さにびっくり・・・。売上や利益が上がっていくのと同時に、こんなにも税金が増えると思っていなかったのでかなりショックでした。

50代雑誌編集（フリーライター）

老後資金はどう準備すればいい？

保険料も抑えたいし、将来にもしっかり備えたい

妻に急かされてほけんの窓口に行くことになりました。二人とも既に保険に入っていたので、行って保険加入を迫られても嫌だし、たまの休日くらいゆっくりしたかったのですが、妻が「どうしても一緒に行ってほしい」というので、内心、渋々行ったのを覚えています。 担当さんは私たちの希望から丁寧に聴いてくださいました。私はちょっとイヤミっぽく「保険料は今より抑えたい」「将来についてもしっかり備えたい」「コロナ禍で収入が不安定になっている」などたくさんのワガママを伝えたところ、担当さんはニッコリしながら私たちの想いを受け止めてくれました。 私はあまり保険が好きではないので、「iDeCoやNISAはどうなの？」とツッコんだところ、担当さんから「ご夫妻にとっては一番合っているかも」と国民年金基金を紹介されました。詳しく話を聴いてみると、収入が不安定な時は口数を減らして掛金の負担を軽減できることや逆に売り上げが良い時は口数を増やして、将来の年金額を増やせることなどを教えてくれました。しかも一切手数料などはかからないとのことでした。ただ、その場ですぐに決断しないで自宅で妻と相談した結果、「1口だけでも始めてみよう」となり申込みました。 ほけんの窓口で保険以外も取扱っているなんてすごいですよね。
